COMO SURGIU A INTERNET ?

[Voltar]

A internet foi criada nos Estados Unidos, na época da Guerra Fria, o departamento de defesa americano queria criar uma rede de comunicação de computadores em potos estratégicos.

A intenção era impedir que as informações valiosas não fossem destruídas pelos os bombardeios inimigos se estivessem localizadas em um único servidor.

Então a ARPA (Advanced Research Projects Agency), uma das subdivisões do Departamento, criou uma rede conhecida por ARPANET, tinha como função interligar laboratórios de pesquisa.

QUAL É A HISTÓRIA DO HTML ?

[Voltar]

O HTML foi criado em 1989, por um cientista chamado Tim Berners-Lee, também conhecido como o pai da Internet.

Foi ele que pegou a linguagem de marcação genérica SGML do Cern e usou como padrão para criar o HTML,

A finalidade inicial era a de tornar possível o acesso e a troca de informações e de documentação de pesquisas, entre cientistas de diferentes universidades.

Ao inventar o HTML ele lançou as fundações da Internet que conhecemos atualmente.

O QUE É O HTML ?

[Voltar]

O HTML é uma Linguagem De Marção, muito utilizado na construção de páginas na web.

E compõe a maior parte das páginas da internet e dos aplicativos online.

HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas.

Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

COMO FUNCIONA O HTML ?

[Voltar]

O HTML é uma linguagem de marcação, essa linguagens são constituídas de códigos que delimitam conteúdos específicos, seguindo uma sintaxe própria.

Os documentos HTML são arquivos que terminam com uma extensão .html ou .htm.

Um navegador lê o arquivo HTML e renderiza o seu conteúdo para que os usuários da internet possam vê-lo.

Todas as páginas HTML possuem uma série de elementos, que consistem num conjunto de tags e atributos.

O QUE SÃO ÁS TAGS HTML ?

[Voltar]

As Tags HTML são elementos usados pra marcar e formatar os conteúdos, as Tags são formadas por uma estrutura própria.

E essas Tags definem o conteúdo e a formatação de um documento HTML.

Elas são usadas para estruturar o conteúdo de um documento, para formatar o texto, para adicionar imagens e outros elementos, etc.

Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Por exemplo: a tag de parágrafo é escrita como < p > e < /p >.

Já para deixar um texto em negrito pode ser usada a tag < Strong > ou < b > no início e < /Strong > ou < /b > no final.

O QUE É UM ELEMENTO HTML ?

[Voltar]

Um Elemento HTML faz parte de uma Página Web, um elemento é qualquer objeto identificável dentro de um documento, por exemplo, um caractere, palavra, imagem, parágrafo ou célula de planilha.

Cada elemento pode ter determinados atributos, na sintaxe HTML, a maioria dos elementos são escritos com marcadores (ainda chamados de etiquetas ou tags) de abertura e um marcador de fechamento, com o conteúdo entre os dois.

O QUE SÃO LISTAS CORDENADAS E LISTAS NÃO CORDENADAS NO HTML ?

[Voltar]

Uma lista não ordenada é uma lista de itens, as listas de itens são marcadas com bullets (tipicamente pequenos círculos pretos).

  • Uma lista não ordenada começa com a tag < ul >.
  • Cada item da lista começa com a tag < li >.
Uma lista ordenada representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.
  • Uma lista não ordenada começa com a tag < ol >.
  • Cada item da lista começa com a tag < li >.
